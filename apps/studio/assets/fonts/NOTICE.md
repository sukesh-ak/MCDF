# Bundled fonts — attribution

MCDF Studio ships these font files (assets, not linked code):

| File | Font | License |
|---|---|---|
| `Roboto-Regular.ttf` | Roboto | Apache-2.0 |
| `RobotoMono-Regular.ttf` | Roboto Mono | Apache-2.0 |
| `fa-solid-900.ttf` | Font Awesome 6 Free (Solid) | SIL OFL 1.1 |

The FontAwesome C++ icon header (`apps/studio/IconsFontAwesome6.h`, from the
IconFontCppHeaders project) is under the zlib license.

Roboto / Roboto Mono are Apache-2.0 (matching the project code license). Font
Awesome Free glyph fonts are OFL-1.1; the "Free" set is used (no Pro glyphs).
